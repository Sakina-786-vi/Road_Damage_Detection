## Overview

AI Road Damage Assessment goes beyond traditional road inspections. Upload a single photo and the system extracts damage features using OpenCV, classifies repair priority with a trained Random Forest model, estimates repair costs, and generates a structured professional report — all in one pipeline.

> **No Gemini API key?** The system falls back to a structured template report automatically.

---

## ✨ Features

- **Drag-and-drop image upload** with instant in-browser preview
- **5-stage CV pipeline** — blur, threshold, edge detection, contour analysis
- **ML priority classification** — Random Forest trained on synthetic damage data
- **Cost estimation** — rule-based formula calibrated to repair priority
- **AI narrative report** — generated by Google Gemini 1.5 Flash via LangChain
- **Privacy-safe** — images are processed in memory and deleted after analysis

---

## 🔬 Analysis Pipeline

| Step | Module | Technology |
|------|--------|------------|
| Image pre-processing | `analyzer.py` | OpenCV — Gaussian blur, adaptive thresholding |
| Feature extraction | `analyzer.py` | Canny edges, Laplacian variance, contour detection |
| Priority classification | `ml_model.py` | Scikit-learn Random Forest (200 trees) |
| Cost estimation | `report_generator.py` | Rule-based formula with priority-based rates |
| Report generation | `report_generator.py` | LangChain + Google Gemini 1.5 Flash |

---

## 📊 Extracted Features

| Feature | Extraction Method |
|---------|-------------------|
| `damage_percentage` | Adaptive thresholding → damaged pixel ratio |
| `num_damaged_regions` | Contour detection with area filter |
| `crack_density` | Canny edge pixel ratio |
| `texture_roughness` | Laplacian variance (normalised) |
| `dark_surface_percentage` | Grayscale intensity threshold |

## 🚀 Quick Start

### Prerequisites

| Requirement | Notes |
|-------------|-------|
| Python 3.10+ | Required |
| Gemini API key | Optional — [get one free](https://aistudio.google.com/app/apikey) |

---

### 1. Enter the project folder

```bash
cd road_damage_app
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv

# macOS / Linux
source venv/bin/activate

# Windows
venv\Scripts\activate.bat
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

```bash
cp .env.example .env
# Open .env and paste your GEMINI_API_KEY
```

### 5. Train the ML model

```bash
python generate_dataset.py   # Generates road_damage_dataset.csv
python ml_model.py           # Trains and saves the Random Forest model
```

### 6. Launch the app

```bash
python app.py
```

Open **http://localhost:5000** in your browser.
